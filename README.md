# Checkout Redirect

Função serverless que redireciona SKUs da Loja Black direto para a pagou.ai.

## Como usar na URL da Lovable

Um produto:
```
https://SEU-PROJETO.vercel.app/api/checkout?skus=TALY0G2P-986
```

Múltiplos produtos:
```
https://SEU-PROJETO.vercel.app/api/checkout?skus=TALY0G2P-986,MYY8GXGX-833&qtds=1,2
```

## Deploy na Vercel

1. Crie um novo projeto na Vercel
2. Faça upload dessa pasta
3. Deploy automático
